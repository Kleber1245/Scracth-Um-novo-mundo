O que são os Blocos de Controle?

No Scratch, o botão "Control" (ou "Controle") é utilizado para gerenciar a execução de scripts e controlar o fluxo do programa.

Principais Blocos de Controle:

  • Quando a bandeira verde for clicada: Inicia a execução do script.
  • Repetir [ ]: Executa um bloco de código um número específico de vezes.
  • Repetir até [ ]: Executa um bloco de código até que uma condição seja verdadeira.
  • Se [ ] então: Executa um bloco de código se uma condição for verdadeira.
  • Se [ ] então senão: Executa um bloco de código dependendo de uma condição.
  • Esperar [ ] segundos: Pausa a execução do script.
  • Esperar até [ ]: Pausa a execução até que uma condição seja verdadeira.
  • Parar [ ]: Interrompe a execução de scripts.
  • Criar clone de [ ]: Cria uma cópia de um sprite.
  • Quando eu começar como clone: Executa código quando um clone é criado.
  • Excluir este clone: Remove um clone.

Essas funções permitem criar lógica complexa e interações dinâmicas em seus projetos.
