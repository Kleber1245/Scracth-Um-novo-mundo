O que são os Blocos de Operadores?

No Scratch, o botão "Operators" (ou "Operadores") é utilizado para realizar operações matemáticas, comparações lógicas e manipulação de texto.

Principais Blocos de Operadores:

  • [ ] + [ ]: Adição.
  • [ ] - [ ]: Subtração.
  • [ ] * [ ]: Multiplicação.
  • [ ] / [ ]: Divisão.
  • número aleatório entre [ ] e [ ]: Gera um número aleatório dentro de um intervalo.
  • > : Maior que.
  • < : Menor que.
  • = : Igual a.
  • e : Operador lógico "E".
  • ou : Operador lógico "OU".
  • não : Operador lógico "NÃO".
  • juntar [ ] com [ ]: Concatena strings de texto.
  • letra [ ] de [ ]: Retorna um caractere específico de uma string.
  • comprimento de [ ]: Retorna o número de caracteres em uma string.
  • [ ] contém [ ]?: Verifica se uma string contém outra substring.
  • resto de [ ] / [ ]: Retorna o resto de uma divisão.
  • arredondar [ ]: Arredonda um número para o inteiro mais próximo.
  • abs de [ ]: Retorna o valor absoluto de um número.
  • piso de [ ]: Arredonda um número para o inteiro menor mais próximo.
  • teto de [ ]: Arredonda um número para o inteiro maior mais próximo.
  • raiz quadrada de [ ]: Calcula a raiz quadrada de um número.
  • seno de [ ]: Calcula o seno de um ângulo (em graus).
  • cosseno de [ ]: Calcula o cosseno de um ângulo (em graus).
  • tangente de [ ]: Calcula a tangente de um ângulo (em graus).
  • arco seno de [ ]: Calcula o arco seno de um valor.
  • arco cosseno de [ ]: Calcula o arco cosseno de um valor.
  • arco tangente de [ ]: Calcula o arco tangente de um valor.
  • ln de [ ]: Calcula o logaritmo natural de um número.
  • log de [ ]: Calcula o logaritmo base 10 de um número.
  • e ^ [ ]: Calcula a exponencial de um número.
  • 10 ^ [ ]: Calcula 10 elevado a um número.

Os blocos de operadores são fundamentais para adicionar lógica e dinamismo aos seus projetos no Scratch, permitindo realizar cálculos, tomar decisões e manipular dados.
