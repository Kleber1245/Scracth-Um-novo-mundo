O que são Meus Blocos?

No Scratch, o botão "My Blocks" (Meus Blocos) é utilizado para criar blocos personalizados, permitindo que os usuários definam suas próprias funções e agrupem códigos.

Funcionalidades de Meus Blocos:

  • Criar um Bloco: Abre a interface para definir um novo bloco personalizado.
  • Nome do Bloco: Define o nome que o seu bloco personalizado terá.
  • Adicionar uma Entrada (Número ou Texto): Permite que o seu bloco receba valores.
  • Adicionar uma Entrada Booleana: Permite que o seu bloco receba valores verdadeiro/falso.
  • Adicionar um Rótulo: Adiciona texto descritivo ao seu bloco.
  • Executar sem atualização da tela: Opção para executar o bloco rapidamente sem redesenhar a tela a cada passo.

Criar seus próprios blocos ajuda a organizar o código, reutilizar sequências de ações e simplificar projetos complexos.
