O que são os Blocos de Aparência?

No Scratch, o botão "Looks" (ou "Aparência") é utilizado para controlar a aparência dos sprites (personagens ou objetos) e como eles se apresentam na tela.

Principais Blocos de Aparência:

  • Dizer [ ] por [ ] segundos: Exibe uma mensagem em uma caixa de fala.
  • Dizer [ ]: Exibe uma mensagem até ser alterada.
  • Pensar [ ] por [ ] segundos: Exibe uma mensagem em uma caixa de pensamento.
  • Pensar [ ]: Exibe uma mensagem de pensamento até ser alterada.
  • Mudar traje para [ ]: Muda a fantasia do sprite.
  • Próximo traje: Muda para a próxima fantasia do sprite.
  • Mudar o cenário para [ ]: Muda o fundo do palco.
  • Próximo cenário: Muda para o próximo fundo do palco.
  • Mudar o efeito [cor/peixe/etc.] por [ ]: Aplica um efeito visual.
  • Definir o efeito [cor/peixe/etc.] para [ ]: Define um valor para um efeito visual.
  • Limpar efeitos gráficos: Remove todos os efeitos visuais.
  • Mudar tamanho por [ ]: Altera o tamanho do sprite.
  • Definir tamanho para [ ]%: Define o tamanho do sprite em porcentagem.
  • Mostrar: Torna o sprite visível.
  • Esconder: Torna o sprite invisível.
  • Ir para a camada [frente/trás]: Controla a ordem de exibição dos sprites.

Essas funções permitem personalizar a aparência dos sprites e criar interações visuais dinâmicas.
