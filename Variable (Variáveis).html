O que são os Blocos de Variáveis?

No Scratch, o botão "Variable" (ou "Variáveis") é utilizado para criar e manipular variáveis, que são espaços de armazenamento para guardar informações que podem mudar durante a execução do seu projeto.

Principais Blocos de Variáveis:

  • Criar uma variável: Abre a interface para criar uma nova variável.
  • Definir [variável] para [ ]: Atribui um valor específico a uma variável.
  • Mudar [variável] por [ ]: Altera o valor de uma variável (soma ou subtrai).
  • mostrar variável [ ]: Torna a variável visível no palco.
  • esconder variável [ ]: Torna a variável invisível no palco.

As variáveis são essenciais para criar projetos interativos e dinâmicos, permitindo armazenar pontuações, controlar estados, e muito mais.
