O que são os Blocos de Eventos?

No Scratch, o botão "Events" (ou "Eventos") é utilizado para definir quando e como os scripts devem começar a ser executados em seus projetos.

Principais Blocos de Eventos:

  • Quando a bandeira verde for clicada: Inicia a execução dos scripts anexados quando o usuário clica na bandeira verde.
  • Quando a tecla [ ] for pressionada: Inicia a execução dos scripts quando uma tecla específica é pressionada.
  • Quando este sprite for clicado: Inicia a execução dos scripts quando o sprite é clicado com o mouse.
  • Quando o cenário mudar para [ ]: Inicia a execução dos scripts quando o fundo do palco muda para um específico.
  • Quando o volume do som > [ ]: Inicia a execução dos scripts quando o volume do som do microfone ultrapassa um certo limite.
  • Receber [ ]: Define um gatilho para iniciar scripts quando uma mensagem específica é recebida.
  • Enviar [ ]: Envia uma mensagem para todos os sprites e o palco.
  • Enviar [ ] e esperar: Envia uma mensagem e pausa o script atual até que todos os scripts que receberam a mensagem terminem.

Os blocos de eventos são cruciais para tornar seus projetos interativos e dinâmicos, permitindo que diferentes partes do seu código respondam a ações do usuário ou a mudanças no ambiente do projeto.
